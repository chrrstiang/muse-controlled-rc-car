# Calibration Data

This directory stores calibration files for the BCI system.

## Files:

- `gyro_neutral.json` - Neutral head position baseline (pitch, roll, yaw)
- `focus_threshold.json` - Personalized focus detection threshold
- `motor_mapping.json` - Steering angle to servo PWM mapping

## Usage:

Run calibration scripts to generate these files before first use.
Recalibrate if system behavior becomes inconsistent.
