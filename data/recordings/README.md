# Data Recordings

This directory stores EEG and gyroscope data recordings for testing and analysis.

## File naming convention:

- `eeg_YYYYMMDD_HHMMSS.csv` - EEG data recordings
- `gyro_YYYYMMDD_HHMMSS.csv` - Gyroscope data recordings
- `session_YYYYMMDD_HHMMSS.log` - Session metadata and events

## Note:

This directory is in .gitignore to avoid committing large data files.
Use for local testing and debugging only.
